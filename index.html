How it works

LOOP’s output is designed to be consistent and readable. The interpreter organizes text into sections that are useful to real users.

1) Extract key obligations

Detects “must / required / prohibited” patterns, bullets, and important lines. Produces a concise list of obligations and scope.

2) Identify ambiguity zones

Flags soft language like “may”, “reasonable”, “subject to”, “material”. This is where interpretation differs across firms and jurisdictions.

3) Format into a trusted structure

Outputs consistent headings: what it says, what it doesn’t say, unclear parts, and watchpoints. Built to reduce misreads.

4) Ready for upgrade

When you want real AI answers + citations, wire the Generate button to a backend endpoint that calls ChatGPT safely (no exposing your API key).
